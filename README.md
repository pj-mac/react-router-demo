## Introduction

This project demonstrates the basic functionality of the React Router.

Start by installing packages:

`npm install`

## Available Scripts

Run in development mode:

`npm start`

Run a production build:

`npm run build`

This builds the app for production to the `build` folder.

Run ESLint:

`npm run lint`
